# Kenya Hardware Pro POS (HPro)
Professional Hardware Management System with eTIMS & M-Pesa Integration.

## 🚀 Vision
A specialized, high-performance POS system designed specifically for the unique inventory and tax requirements of Kenyan hardware stores.

## 🛠 Features
- **Smart Inventory**: Construction materials, batch tracking (e.g., cement expiry), and unit conversions (m, ft, kg).
- **Tax Compliance**: Built-in **KRA eTIMS** integration for automated invoice reporting.
- **Financial Gateway**: Integrated **M-Pesa Stk Push** and Paybill reconciliation.
- **Credit Management**: Track customer debts (Lupa) and supply chain payments.
- **Reporting**: Daily sales, profit margin analysis, and stock levels.

## 🏗 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4 (Premium UI)
- **State**: Zustand (Local & Persistence)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 🚦 Getting Started
1. `npm install`
2. `npm run dev`
